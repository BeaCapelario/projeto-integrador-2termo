Este repositório documenta o desenvolvimento do Projeto Integrador referente ao segundo termo do curso. O projeto visa a aplicação prática e interdisciplinar dos conceitos aprendidos, unindo o desenvolvimento front-end e back-end para solucionar um problema real ou simular um ambiente de produção completo.

---

### O que foi explorado:
* Integração Full Stack: Conexão entre a interface do usuário e a lógica de servidor.
* Persistência de Dados: Gerenciamento de informações através de sistemas de arquivos ou bancos de dados.
* Arquitetura MVC: Organização do projeto seguindo padrões de modelos, visões e controladores.
* Experiência do Usuário (UX): Desenvolvimento de uma interface intuitiva e funcional.
* Trabalho Colaborativo: Práticas de controle de versão e documentação para alinhamento de requisitos.

---

### Tecnologias Utilizadas
| Tecnologia | Descrição |
| :--- | :--- |
| Node.js | Ambiente de execução para o back-end da aplicação. |
| Express.js | Framework para construção de rotas e APIs. |
| JavaScript | Linguagem utilizada em toda a stack do projeto. |
| HTML5 / CSS3 | Estruturação e estilização da interface front-end. |
